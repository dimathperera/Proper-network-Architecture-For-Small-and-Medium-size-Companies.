**HND Cybersecurity & Network Architecture Final Year Project**

**Candidate:** 
**Standards:** ISO 27001 & SFIA Skills Framework

## 📋 Project Overview

This project develops a localized "Security-by-Design" framework 
for Sri Lankan startups and SMEs, providing standardized, 
affordable network architecture aligned with international standards.
